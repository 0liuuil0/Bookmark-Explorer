一个现代化的、功能完整的收藏夹管理工具，采用三栏设计，支持导入、导出、搜索和管理浏览器收藏夹。

苦整理edge收藏夹久矣，找遍全网没发现可以整理edge收藏夹的工具，只发现大佬lenycool写的收藏夹管理增强扩展，但使用起来无法通过拖动整理，遂自力更生开发出这个。

## ✨ 功能特性

### 📁 三栏布局设计
- **左侧栏**：文件夹树状结构，只显示文件夹
- **中间栏**：链接网格视图，显示当前文件夹内容
- **右侧栏**：操作面板，提供各种管理功能

### 🔄 核心功能
- ✅ **导入/导出**：支持Edge/Chrome导出的HTML格式
- ✅ **树状层级文件夹**：直观的文件夹结构展示
- ✅ **智能搜索**：在当前文件夹内快速搜索链接
- ✅ **多选操作**：支持多种选择模式
- ✅ **批量移动**：将选中的项目移动到其他文件夹
- ✅ **拖拽操作**：拖拽链接进行移动
- ✅ **新建管理**：创建新文件夹和链接
- ✅ **实时统计**：显示收藏夹统计数据
- ❌**未实现功能**（后续待开发）：删除文件夹、更清晰的文件夹层级

### 🎯 选择模式
- **Ctrl+点击**：多选/取消选择
- **Shift+点击**：选择连续范围
- **Ctrl+A**：全选当前文件夹
- **鼠标拖拽**：框选多个项目
- **Delete键**：删除选中项目
- **Esc键**：取消所有选择

### 🎨 界面特色
- 现代化UI设计
- 响应式布局，支持不同屏幕尺寸
- 实时预览和统计信息
- 直观的分栏导航

## 🚀 快速开始

### 直接运行
1. 下载项目文件到本地
2. 打开 `index.html` 文件（比如使用VScode）
3. 开始使用！
